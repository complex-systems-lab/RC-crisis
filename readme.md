# Dynamics of Reservoir Computing for Crises Prediction

This repository contains the Python code and Jupyter notebooks to generate the results of the paper:

**Dynamics of Reservoir Computing for Crises Prediction**  
https://doi.org/10.1103/8b7z-fnjd

## Overview

The project demonstrates how reservoir computing (RC) can predict discontinuous transitions such as crises in dynamical systems. It includes two Jupyter notebooks:

- `Logistic_map_scaling.ipynb`  
  Reproduces the prediction of the scaling exponent by RC for the logistic map.

- `Logistic_map_crisis.ipynb`  
  Reproduces the correct prediction of the mechanism underlying crisis phenomena.

## Dependencies

The notebooks require the following Python packages:

- [NumPy](https://numpy.org/)  
- [Matplotlib](https://matplotlib.org/)  
- [NetworkX](https://github.com/networkx/networkx)  
- [Directional Fibers](https://github.com/garrettkatz/directional-fibers)
